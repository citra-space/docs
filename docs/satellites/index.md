# Satellites

The Satellites section provides tools for browsing, filtering, and analyzing objects in Earth orbit. Citra Space aggregates satellite data from multiple providers into a unified catalog, giving you access to orbital elements, transmission frequencies, aliases, and observation data.

From here you can:

- **Browse and filter** the satellite catalog by name, NORAD ID, country, and other attributes
- **View detailed profiles** for individual satellites, including aliases and transmission data
- **Analyze orbital elements** with history charts, comparison tools, and residual analysis
- **Calculate ground station visibility** to plan observation windows
- **Create and manage element sets** for satellites you track
