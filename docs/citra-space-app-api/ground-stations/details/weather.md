# Weather

The Weather section on the ground station detail page provides real-time weather data, forecasts, and astronomical information for the station's geographic location. This data supports observation planning by helping you assess current and upcoming conditions.

## Current Conditions

The current weather card displays live conditions at the ground station location, including:

- **Temperature** - Current temperature reading
- **Wind** - Wind speed and direction
- **Conditions** - General weather description (clear, cloudy, rain, etc.)

## Weather Map

An interactive map centered on the ground station's location provides a visual overview of regional weather patterns. Use the map to assess cloud cover and weather systems in the area surrounding the station.

## Hourly Forecast

The hourly forecast provides a 48-hour prediction of weather conditions at the ground station. Use this view to identify upcoming clear-sky windows for optical observations or assess conditions for planned satellite passes.

## 8 Day Forecast

The extended daily forecast covers 8 days, showing high and low temperatures, precipitation probability, and general conditions for each day. This longer-range view supports advance scheduling of observation campaigns.

## Sun and Moon

The Weather section also includes sun and moon data for the ground station's location:

| Data | Description |
|------|-------------|
| **Sunrise** | Time the sun rises above the horizon |
| **Sunset** | Time the sun sets below the horizon |
| **Moonrise** | Time the moon rises above the horizon |
| **Moonset** | Time the moon sets below the horizon |
| **Moon Phase** | Current phase of the moon (new, crescent, quarter, gibbous, full) |

{: .note }
> Moon phase data is particularly useful for optical observations. A bright moon increases sky background noise, which can reduce the effectiveness of optical satellite tracking.

## Use Cases

Weather monitoring supports:

- **Observation planning** - Identifying clear-sky windows for optical satellite tracking
- **Cloud cover assessment** - Evaluating whether conditions are suitable for planned observations
- **Schedule optimization** - Using forecasts to schedule observation sessions during favorable conditions
- **Go/no-go decisions** - Quickly checking current conditions before initiating an observation session
