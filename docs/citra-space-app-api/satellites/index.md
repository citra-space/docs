# Satellites

The Satellites section provides tools for browsing, filtering, and analyzing objects in Earth orbit. Citra Space aggregates satellite data from multiple providers into a unified catalog, giving you access to orbital elements, transmission frequencies, aliases, observation data, and more.

From here you can:

- [**Browse and filter**](browsing-and-filtering) the satellite catalog by name, NORAD ID, country, and other attributes
- [**View aliases**](details/aliases) and [**transmissions**](details/transmissions) for individual satellites
- [**Analyze orbital elements**](details/orbital-elements) with history charts, comparison tools, and [**residual analysis**](details/observation-residuals)
- [**Detect maneuvers**](details/maneuvers) — see deliberate orbital changes with delta-v breakdowns across radial, in-track, and cross-track components
- [**Calculate close approaches**](details/close-approaches) between a satellite and other cataloged objects for conjunction screening
- [**Calculate ground station visibility**](details/ground-station-visibility) to plan observation windows across your stations
- [**Create and manage element sets**](details/element-sets) including GP, external, and analyst-derived orbital elements
- [**Request collections**](details/collection-requests) — schedule optical or TDOA observation windows across available sensors
- [**View tasking schedules**](details/tasking-schedule) — track scheduled, completed, and canceled observation tasks
- [**Review image uploads**](details/image-uploads) and [**RF captures**](details/rf-captures) — access processed telescope imagery and radio frequency recordings from ground-based sensors
