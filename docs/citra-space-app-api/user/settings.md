# Settings

The Settings page lets you customize your interface preferences. Changes are synced to your account and apply across all devices.

## Understanding Settings

Interface settings control the visual appearance and feature availability of the Citra Space app. These are personal preferences tied to your account, not shared with your group or organization.

## Display Mode

Display Mode controls the color theme of the interface. Choose from the following options:

| Option | Description |
|--------|-------------|
| **Light** | Light background with dark text |
| **Dark** | Dark background with light text |
| **System** | Automatically follows your operating system's appearance preference |

Select the radio button next to your preferred display mode. The change takes effect immediately.

## Tactical Mode

Tactical Mode enables Counter-ISR tools and mission-focused navigation elements throughout the app. When enabled, additional analysis features become available in the satellite and sensor views.

{: .note }
> Tactical Mode is only available for Government tier subscriptions and above. If your account does not meet the tier requirement, the Tactical Mode toggle will not appear on the Settings page.
