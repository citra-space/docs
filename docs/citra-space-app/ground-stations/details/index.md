# Details and Tools

When you select a ground station from the list, the detail page displays the station's location information along with tools for satellite pass calculation, sensor management, and weather monitoring.

The detail page is organized into the following sections:

- **[Find Satellite Passes](find-satellite-passes)** - Calculate when satellites will be visible from this location
- **[Sensors](sensors)** - Manage telescopes and antennas at the station
- **[Weather](weather)** - View current conditions, forecasts, and sun/moon data for observation planning
